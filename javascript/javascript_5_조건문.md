### 짧은 조건식(p.89)

* A || B ⇐ A 또는 B, 둘 중 하나가 참이면 참 = A와 B, 둘 모두 거짓이어야 거짓
* A && B ⇐ A와 B, 둘 모두 참이어야 참 = A 또는 B 둘 중 하나가 거짓이면 거짓

![image-20200123152610361](images/image-20200123152610361.png)

​		=> 좌변이 참이변 우변을 무시, 좌변이 거짓이므로 우변이 참인지 거짓인지 검사
